This Repositor contains 2 files.

'Module 2 Challenge VBA Script File.bas' and 'Module 2 Challenge Screenshots.docx'

'Module 2 Challenge VBA Script File.bas' is intended to run with the Multiple_year_stock_data.xlsx workbook that was provided for the Module 2 Challenge. 

It consists of 3 Subroutines.  The first subroutine named RunMacro() calls on the 2nd and 3rd subroutines. Run the first subroutine as needed to replicate results.

'Module 2 Challenge Screenshots.docx' contains screenshots of the worksheets Q1, Q2, Q3, and Q4 within workbook Multiple_year_stock_data.xlsx after the VB code has ran.
